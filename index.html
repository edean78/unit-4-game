Click a Crystal to Begin

Instructions

When the game begins, a Target Score will be randomly selected and displayed.

There are four crystal below. The value of each crystal is randomly selected but only by clicking the crystal will the value be revealed.

Each time you click on a crystal the value will be added to the previously clicked crystal.

The player wins the game by clincking the crytals in a way that the sum equals the Target Score. If the players score goes above the Target Score.

After each game the Target Score and the value of each crystal will change.
